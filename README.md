# Sauna Sokolec — strona statyczna

Gotowa strona HTML/CSS do publikacji na Cloudflare Pages.

## Jak edytować ceny
Otwórz `index.html` i znajdź sekcję `id="cennik"`.

## Jak zmienić zdjęcia
Podmień pliki w folderze `assets`, zachowując nazwy plików.

## Publikacja
1. Utwórz repozytorium na GitHub.
2. Wgraj wszystkie pliki z tego folderu.
3. Cloudflare Pages → Create project → Connect to Git.
4. Framework preset: None.
5. Build command: zostaw puste.
6. Output directory: `/`.

## Edycja przez AI / Codex
Po podłączeniu repozytorium możesz prosić Codex np.:
- „Zmień cenę dla 4 osób na 189 zł”
- „Dodaj sekcję voucher prezentowy”
- „Podmień numer telefonu”
- „Dodaj formularz rezerwacji”
